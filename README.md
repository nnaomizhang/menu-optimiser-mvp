# Menu Optimiser MVP

An AI-powered restaurant menu optimisation tool built with Streamlit and GPT-4.

## Setup

1. Clone the repo
2. Install dependencies:
pip install -r requirements.txt

3. Add your OpenAI API key to `.env`:
OPENAI_API_KEY=sk-your-key-here

4. Run the app:
streamlit run app.py

## Usage
- Upload your menu CSV file or create manually
- Follow the 4-step pipeline to validate, analyse, get recommendations, and download your personalised report

## Required CSV Columns
- item_name
- category
- current_price
- food_cost
- monthly_units_sold
